The pack is simple:
  • modded Console2
  • modded Far2
  • modded FarColorer2
The basic idea is to implement a shared buffer with extended annotation attributes (like true color) for each character.
Then, Console2 learns how to display this, Far2 learns how to draw this, and Colorer just learns that it now can use RGB styles!
Simple, huh?
